# Changelog
All notable changes to the Demand Pattern Taxonomy will be documented in this file.

- Quantitative Formula Reference for all 5 granularities (Daily, Weekly, Monthly, Quarterly, Yearly).
- Professional MkDocs Material documentation portal with dark mode and MathJax support.
